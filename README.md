# Paradise Nursery Plant Shop

An elegant, fully responsive React & Redux full-stack e-commerce application designed for plant enthusiasts. Users can explore various categories of indoor and outdoor plants, filter through items, view detailed breakdowns of their shopping cart, and seamlessly manage item quantities.

## Key Features
- **Dynamic Catalog:** Explore distinct plant categories like Air Purifying and Medicinal.
- **Redux State Management:** Smooth cart interactions powered by Redux Toolkit slices.
- **Intuitive Shopping Cart:** Real-time calculation of total quantities and pricing configurations.
